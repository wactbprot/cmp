conds-match?

(conds-match? k)

Gathers all information for the given definitions key comparison and checks all of the found conditions.

(conds-match? "ref@definitions@0@class")
;; false
(conds-match? "ref@definitions@1@class")
;; true

get-comp-val

multimethod

Returns the compare value belonging to key k. If the keyword kw is not nil it is used to extract the related value.

(get-comp-val "ref@definitions@0@cond@0" :Value)
;; "Pa"
