devhub!

(devhub! pre-task)

Param is called pre-task because some tasks come with a :PreScript which has to be executed in order to complete the task (sometimes the :Value is computed by the :PreScript).

 (devhub! {:Action "TCP" :Port 23 :Host "localhost" :Value "Hi!"})

(resolve-pre-script task)

Checks if the task has a :PreScript (name of the script to run) and an :Input key. If not, task is returned.
