clj->val

(clj->val x)

Casts the given (complex) value x to a writable type. json is used for complex data types.

Example:

(clj->val {:foo "bar"})
;; "{"foo":"bar"}"
(clj->val [1 2 3])
;; "[1,2,3]"

lp

(lp i)(lp i n)

Left pad the given number if it is not a string. Default is (cfg/key-pad-length (cfg/config)).

Example:

 (u/lp 2)
;; "002"
(u/lp "02")
;; "002"
(u/lp 2)
;; "002"
(u/lp true)
;; "000"
(u/lp "003")
;; "003"
(u/lp "000003")
;; "003"

val->clj

(val->clj v)

Parses value v and returns a clojure type of it.

(val->clj "-1e-9")
;; -1.0E-9
;; class:
;;
(class (val->clj "1.23"))
;; java.lang.Double
(class (val->clj "a"))
;; java.lang.String
(class (val->clj "[]"))
;; clojure.lang.PersistentVector
(class (val->clj "{}"))
;; clojure.lang.PersistentArrayMap
(class (val->clj "{"a":1}"))
;; clojure.lang.PersistentArrayMap
