append-if

(append-if m k v)

Appends v to the value of k. If k does not exist in m, k [v] is assoced. If k does exist in m, v is conjed.

(append-if {:Value [1 2 3]} :Value 4)
;; {:Value [1 2 3 4]}

clj->val

(clj->val x)

Casts the given (complex) value x to a writable type. json is used for complex data types.

Example:

(clj->val {:foo "bar"})
;; "{"foo":"bar"}"
(clj->val [1 2 3])
;; "[1,2,3]"

val->clj

(val->clj v)

Parses value v and returns a clojure type of it.

(val->clj "-1e-9")
;; -1.0E-9
;; class:
;;
(class (val->clj "1.23"))
;; java.lang.Double
(class (val->clj "a"))
;; java.lang.String
(class (val->clj "[]"))
;; clojure.lang.PersistentVector
(class (val->clj "{}"))
;; clojure.lang.PersistentArrayMap
(class (val->clj "{"a":1}"))
;; clojure.lang.PersistentArrayMap
