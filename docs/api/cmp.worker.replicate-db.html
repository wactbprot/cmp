cmp.worker.replicate-db

replicate!

(replicate! task)

Replicate a couchdb database by posting:

{
  "_id": "my_rep",
  "source": "http://myserver.com/foo",
  "target":  "http://user:pass@localhost:5984/bar",
  "create_target":  true,
  "continuous": true
}

to the /_replicate endpoint.
